# Auth0 iOS Swift - Native Social Login Samples

This is a collection of samples that demonstrate how to integrate native social Identity Providers with Auth0.

## Available samples

* [SIWA](/00-login-siwa) - Sign in With Apple sample. [Tutorial](https://auth0.com/docs/quickstart/native/ios-swift-siwa)
* [Facebook](/00-login-facebook) - Facebook Login sample. [Tutorial](https://auth0.com/docs/quickstart/native/ios-swift-facebook-login)
